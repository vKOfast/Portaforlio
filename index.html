Mi Trayectoria

Soy Abraham Jacobo Marroquin Martínez, tengo 25 años y soy un desarrollador de software con experiencia en la creación de aplicaciones empresariales y soluciones tecnológicas.

Mi especialidad es el desarrollo con C# y .NET, con amplia experiencia en SQL Server para la gestión de bases de datos. He trabajado en diversos proyectos donde he aplicado estos conocimientos para crear soluciones robustas y eficientes.

También tengo experiencia en Python para scripting y automatización, así como conocimientos en JavaScript, HTML y CSS para el desarrollo frontend cuando es necesario.

Me considero un profesional comprometido, con capacidad para aprender rápidamente y adaptarme a nuevos desafíos. Siempre busco oportunidades para crecer y aportar valor a los proyectos en los que participo.
